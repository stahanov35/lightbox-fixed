JCF

Особенности:

  • Полностью модульная библиотека с различным набором которые легко побключить указав значение при вызове
  • Удачно подобранные опции позволяют решить максимум задач без необходимости что-то менять в коде модулей
  • Плохо дружит с обилием динамики (могут вылазить баги и в ие7 из-за этого)

Choosen

Особенности:

  • Возможность отображения выбраных опций мультиселекта в строку (удаление по кнопке close)
  • Полностью нативный селект
  • Хорошая настройка, документация.
  • Легко менять структуру кастом селекта

Uniform

Особенности:

  • Возможность кастомить input[type=range]
  • Нет кастом дропа
  • Встроенная валидация
  • Нет возможности кастомить дроп
Githab:
https://github.com/pixelmatrix/uniform
Licensed:
MIT License

jClever

Особенности:

  • Есть набор пользовательских методов с помощью которых можно управлять поведением элементов, добавлять новые, делать активными, неактивными элементы и др.
  • Присутствует метод destroy, который отменяет действие плагина. Поля формы возвращаются в исходное состояние
  • Возможность отображения выбраных опций мультиселекта в строку (нет возможности удалять по клику)
  • Встроенная валидация
  • Disabled состояние перебивается фокусом, хотя выбрать другой пункт невозможно, но складывает впечатление что селект не работает
  • Необходимо подключать дополнительные библиотеки (jquery.mousewheel.js, jquery.jscrollpane.min.js)
Githab:
https://github.com/denis-zavgorodny/jClever
Documentation:
http://tools.alterego-russia.ru/jclever/#docs-user-options

jQuery-SelectBox

Особенности:

  • Можно генерировать любую структуру опциий в дропе для каждого пункта инлайново на option в атрибуте alt = "структура"

  • Дроп генерится рядом с select-ом, но при открытии переносится в конец тега, который можно указать в вызове примерно так .sb({ ddCtx: function() { return $(this).closest("tag_name");
  • Не отображается скролл в дропе на мобилах, контент можно проскролить, но понять что там есть еще пункты меню которые можно проскролить - не понятно
  • Так как отсутствует умное позиционирование дропа, то при его раскрытии в самой низу страницы он или обрежется или, если нет overflow: hidden и появляется скролл, то соответственно позиционирование дропа смещается влево
Githab:
https://github.com/revsystems/jQuery-SelectBox
